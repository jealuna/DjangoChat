# DjangoChat
Implementación Simple de un Chat usando Django y React

- [Instalación](#Instalacion)
- [Ejecución](#Ejecucion)

# Instalacion
```bash
$ docker-compose build
```
# Ejecucion

```bash
$ docker-compose up --force-recreate
```
En el navegador abrir la url:
```bash
http://localhost:3000/
```
